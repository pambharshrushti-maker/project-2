# 📸 Image Slider (Carousel)

A responsive **Image Slider (Carousel)** built using **HTML, CSS, and JavaScript**. This project allows users to browse images using **Next** and **Previous** buttons, automatic sliding, image counter, and navigation dots.

---

## 🚀 Project Overview

The Image Slider is a beginner-friendly JavaScript project that demonstrates DOM manipulation, arrays, functions, events, and timers. It provides a smooth image browsing experience with both manual and automatic navigation.

---

## ✨ Features

* 🖼️ Display 5 Images
* 👀 Show One Image at a Time
* ▶️ Next Button
* ◀️ Previous Button
* 🔄 Infinite Loop Slider
* ⏱️ Auto Slide Every 3 Seconds
* 📍 Navigation Dots
* 🔢 Image Counter (Example: Image 2 of 5)
* 📱 Responsive Design

## ⚙️ How It Works

1. The image paths are stored in a JavaScript array.
2. Initially, the first image is displayed.
3. Clicking the **Next** button shows the next image.
4. Clicking the **Previous** button shows the previous image.
5. When the last image is reached, the slider starts again from the first image.
6. When the first image is reached and Previous is clicked, it moves to the last image.
7. The image changes automatically every 3 seconds.
8. Navigation dots indicate the current image.
9. The counter displays the current image number.
